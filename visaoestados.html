Unified View of States Based on Indicators

Develop a practical example of data visualization in Power BI, applying various features such as API data connections, data transformation, aggregations, and data cleaning using Power Query. Optimize visualizations for better understanding and maintenance while leveraging DAX measures with specific contexts. The data used comes from open sources provided by official Brazilian entities such as IBGE and IPEA.

Main concepts

Project Objective

  • Develop a single dashboard displaying key indicators (social, economic, financial, education, health, and security) per state, showing trends and rankings.
  • Ensure a connected and easily updatable database using official Brazilian data sources such as IBGE and IPEA.

Model Structure

  • Consolidate all indicators into a single table.
  • Create auxiliary tables to support ranking calculations and indicator consolidation.
  • Ensure all data dynamically updates based on state abbreviations (UF codes).

Measures & Calculations

  • Selection control for filters.
  • Indicator trend analysis.
  • Latest available values.
  • State ranking based on weighted indicator comparisons.
